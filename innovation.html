Innovation & Projects

  • Land Use and Land Cover (LULC) Classification using Sentinel-1 SAR Data

    A Google Earth Engine (GEE) project that classifies land use and land cover for Kisumu County, Kenya, using Sentinel-1 Synthetic Aperture Radar (SAR) data. The project demonstrates how radar imagery can be used for LULC mapping, especially useful in areas with persistent cloud cover where optical imagery is limited.
